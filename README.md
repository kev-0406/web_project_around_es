# Nombre del proyecto:

Around The U.S.

## Descripción:

Se trata de una pagina web interactiva donde los usarios pueden compartir mediante fotos de paisajes su viaje alredor de los Estados Unidos.

## Funcionalidad:

El usuario puede editar su perfil, agregar tarjetas nuevas, dar "me gusta", eliminar paisajes y abrir las imagenes en pantalla completa.

## Tecnologias usadas:

- HTML5
- CSS3
- JavaScript puro
- Metodologia BEM

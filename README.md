# Nombre del proyecto:

Around The U.S.

## Descripción:

Se trata de una página web interactiva donde los usuarios pueden compartir mediante fotos de paisajes su viaje alrededor de los Estados Unidos.

## Funcionalidad:

- El usuario puede editar su perfil y agregar tarjetas nuevas.
- Funcionalidad de dar "me gusta" y eliminar tarjetas.
- Las imagenes se pueden abrir en pantalla completa.
- Validación de los formularios en vivo para garantizar que los datos ingresados sean correctos.
- Mejora en la experiencia de usuario (UX): las ventanas emergentes se pueden cerrar haciendo clic en la superposición o presionando la tecla Esc.

## Tecnologias usadas:

- HTML5
- CSS3
- JavaScript puro (ES6)
- Arquitectura de JavaScript Modular
- Metodología BEM
